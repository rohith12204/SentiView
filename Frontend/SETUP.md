# Sentiview Frontend Setup

## Install & Run
```bash
cd Frontend
npm install
npm run dev
```

App runs on: http://localhost:5173

## Key Features
- **Home** — Hero, search, trending products preview
- **Search** — Product details, AI summary, aspects, add to cart
- **Trending** — Filter by brand/rating/sentiment, add to cart
- **Compare** — Pick 2-3 products, side-by-side aspect comparison
- **Sentiment Timeline** — 6-month trend charts (real API)
- **Insights Dashboard** — KPIs, donut chart, keywords, top products
- **Cart** — Manage items, qty controls, order summary
- **Payment** — Dummy card payment modal with animation
- **Admin** — Product CRUD, order management (admin users only)
- **Auth** — Sign in / sign up with token persistence
